# Editing Consent Information for Custom Applications
Application administrators can customize the information that appears in the OAuth consent page for applications.
If your application's resources is configured to require consent, then IAM provides a consent page in which users must allow to access the application's resources. By default, this consent page is branded with IAM information, but you can customize the information that appears in the page.
  1. Open the **navigation menu** and select **Identity & Security**. Under **Identity** , select **Domains**. 
  2. Click the name of the identity domain that you want to work in. You might need to change the compartment to find the domain that you want. Then, click **Integrated applications**.
  3. Select the application that you want to modify.
  4. Select **Consent information**.
  5. Provide values for the fields that you want to customize, and then select **Save changes**.
